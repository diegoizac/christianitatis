# Botão

## Descrição

O componente Botão é um elemento fundamental da interface, projetado para ser altamente reutilizável, acessível e personalizável.

## Uso Básico

```tsx
import { Button } from '@/components'

export default function Example() {
  return <Button variant="primary">Clique Aqui</Button>
}
```

## Variações

O botão possui três variações principais:

### Botão Primário

Usado para ações principais e de maior destaque na interface.

<Primary />

### Botão Secundário

Para ações secundárias ou alternativas.

<Story of={ButtonStories.Secundário} />

### Botão com Contorno

Ideal para ações terciárias ou quando se deseja um visual mais sutil.

<Story of={ButtonStories.Contorno} />

## Acessibilidade

O componente Button foi desenvolvido seguindo as melhores práticas de acessibilidade:

- ✅ Suporte completo a navegação por teclado
- ✅ Estados focáveis claramente visíveis
- ✅ Atributos ARIA apropriados
- ✅ Alto contraste nas cores
- ✅ Feedback visual ao interagir

## Boas Práticas

- Use o botão primário apenas para a ação principal da tela
- Mantenha textos curtos e objetivos
- Evite usar mais de um botão primário por seção
- Prefira o botão secundário para ações destrutivas
- Use o botão outline para ações opcionais

Cenários importantes para teste:
